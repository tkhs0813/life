# デイリーダイジェスト生成

RSSフィードから注目記事を収集し、日本語のMarkdownダイジェストとして保存する。

## 実行手順

### Step 1: 準備

1. 今日の日付を確認する
2. `digests/YYYY-MM-DD.md` が既に存在するか確認する。存在する場合はユーザーに上書きするか確認する
3. `digests/` ディレクトリが存在しない場合は作成する

### Step 2: フィード取得

1. リポジトリルートの `feeds.yml` を読み込む
2. 各フィードのURLに対して WebFetch でRSSフィードを取得する
3. 取得したXMLから最新の記事タイトル・URL・概要を抽出する
4. フィード取得に失敗した場合はスキップして次のフィードに進む

### Step 3: 記事の選定

全フィードの記事から注目すべきものをピックアップする。選定基準:

- **新規性**: 今日〜直近数日の記事を優先
- **影響度**: 業界全体に影響がありそうなニュース
- **注目度**: コメント数やリアクションが多い話題

合計20件程度を目安に選定する。

### Step 4: ダイジェスト生成

このスキルと同じディレクトリにある [template.md](template.md) のフォーマットに従い、日本語でダイジェストを作成する。

- 各記事のタイトルはリンク付きで記載する
- 要約は1-3行で簡潔に日本語で書く
- 記事の内容からテーマ別にグルーピングして整理する
- 最後に「ひとこと」で全体傾向をまとめる

### Step 5: 保存

1. `digests/YYYY-MM-DD.md` としてWriteツールで保存する
2. 保存完了を報告し、ピックアップした記事数を表示する

### Step 6: コミット＆プッシュ

1. 生成したダイジェストファイルを `git add` する
2. コミットメッセージ: `digest: YYYY-MM-DD` でコミットする
3. `origin main` にプッシュする

## 注意事項

- 出力は全て日本語で行う
- リンクは元記事のURLをそのまま使う
- 確認できない情報は含めない
- フィード取得エラーは静かにスキップする
