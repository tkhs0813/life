# ダイジェスト出力テンプレート

以下のフォーマットに従ってダイジェストを生成する。テーマ別の見出しは記事の内容に応じて自由に設定してよい（例: 「AI・LLM」「フロントエンド」「OSS・開発ツール」「セキュリティ」など）。

---

```
# デイリーダイジェスト: {YYYY-MM-DD}

## {テーマ1}

### [{記事タイトル}]({URL})
- **ソース**: {フィード名}
- {1-3行の日本語要約}

### [{記事タイトル}]({URL})
- **ソース**: {フィード名}
- {要約}

## {テーマ2}

### [{記事タイトル}]({URL})
- **ソース**: {フィード名}
- {要約}

...

## ひとこと

{今日の全体的な傾向や注目ポイントを1-2文でまとめる}

---
*{YYYY-MM-DD} 生成*
```

## ルール

1. 見出しレベル: H1=日付、H2=テーマ、H3=個別記事
2. 記事タイトルはMarkdownリンクにする
3. 各記事の要約は簡潔に（最大3行）
4. 「ひとこと」で全体の傾向を俯瞰する
